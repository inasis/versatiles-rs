# Versatiles

Versatiles is a Rust-based project for processing and serving tile data.

## Installation

1. **Clone the Repository**:
	```sh
	git clone https://github.com/versatiles-org/versatiles-rs.git
	cd versatiles-rs
	```

2. **Build the Project**:
    ```sh
    cargo build
    ```
